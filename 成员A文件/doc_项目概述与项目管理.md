## 第一章 项目概述

### 1.1 项目背景

单细胞转录组学（scRNA-seq）技术的快速发展产生了海量细胞基因表达数据。以本项目使用的肝脏数据集（liver.h5ad）为例，包含 **69,032 个细胞、32,397 个基因**的测量值，数据规模庞大，人工分析效率极低。

研究人员需要能够快速检索"与给定细胞最相似的细胞群体"，用于：
- 辅助细胞类型注释
- 发现疾病相关细胞亚群
- 跨数据集比对细胞状态

传统线性扫描（暴力搜索）在 69K+ 细胞规模下单次查询耗时约 **1.5ms**，当数据量增至百万级时将严重影响用户体验。本项目旨在构建一套基于 **近似最近邻（ANN）算法** 的高性能单细胞检索系统，将查询耗时压缩到 **0.3ms 以内**，同时保持 **99% 以上的召回率**。

### 1.2 系统目标

| 目标类型 | 具体要求 |
|---------|---------|
| **功能目标** | 支持 Top-K 相似细胞检索、条件过滤检索、多数据集联合检索 |
| **性能目标** | 单次查询 ≤ 0.5ms（69K 细胞，30 维 PCA，CPU 环境） |
| **可用性目标** | 系统提供 Web 界面，支持向量上传查询和可视化展示 |
| **扩展性目标** | 支持动态添加/删除数据集，无需重启服务 |

### 1.4 技术选型

| 模块 | 技术选型 | 选型理由 |
|------|---------|---------|
| ANN 索引（精度优先） | **FAISS IVF**（Facebook AI） | 召回率接近 100%，构建速度快（0.06s/69K），工业级稳定 |
| ANN 索引（速度优先） | **HNSW**（hnswlib） | 查询速度最快（0.2ms），图结构天然支持增量插入 |
| 数据格式 | **NumPy .npy** | 读取速度快，与 Python 生态无缝对接 |
| 后端框架 | **Flask** | 轻量、学习成本低，适合原型开发 |
| 数据预处理 | **Scanpy** | 单细胞数据分析标准工具，API 成熟 |
| 容器化 | **Docker** | 环境一致性，方便结项提交和评分复现 |

### 1.5 数据集说明

| 属性 | 值 |
|------|-----|
| 数据集名称 | liver.h5ad（肝脏单细胞转录组数据） |
| 细胞总数 | 69,032 |
| 基因总数 | 32,397 |
| 细胞类型 | 19 种（肝细胞、内皮细胞、T细胞等） |
| 向量维度 | 30（自带 PCA）/ 64（重新计算） |
| 文件格式 | AnnData .h5ad |
| 关键元数据字段 | cell_type, disease, AgeGroup, sex, donor_age |

### 1.6 性能指标（实测）

> 测试环境：CPU，Python 3.14，数据集 69,032 细胞 × 30 维 PCA，Top-K=10，随机 200 条查询

| 方法 | 索引构建耗时 | 平均查询耗时 | 召回率@10 | 内存占用 |
|------|------------|------------|----------|---------|
| 暴力搜索（基准） | — | ~1.5ms | 100%（基准） | 低 |
| **FAISS IVF**（推荐） | ~0.06s | **~0.3ms** | **~100%** | 低 |
| **HNSW** | ~0.53s | **~0.2ms** | **~99%** | 中 |

### 2.4 模块接口约定

#### A ↔ B 接口（向量格式）

```python
# 成员 B 输出（cell_vectors.npy）
vectors: np.ndarray  # shape=(69032, 30), dtype=float32, L2归一化

# 成员 A 消费
mgr = IndexManager("../成员B文件/cell_vectors.npy")
```

#### A ↔ C 接口（IndexManager）

```python
# 成员 C 调用（Flask 启动时初始化一次）
from index_builder import IndexManager

index_mgr = IndexManager("../成员B文件/cell_vectors.npy")
index_mgr.load("../成员A文件/index_store")

# 单次查询
distances, indices = index_mgr.search(query_vec, k=10)
# distances: (1, k) float32，余弦相似度，越大越相似
# indices:   (1, k) int64，细胞全局索引，与 cell_metadata.csv 行号对应

# 状态查询（供 /api/index/status 接口）
status = index_mgr.status()

# 数据集管理（供 /api/dataset 接口）
index_mgr.add_dataset("kidney", "kidney_vectors.npy")
index_mgr.remove_dataset("kidney")
datasets = index_mgr.list_datasets()
```
