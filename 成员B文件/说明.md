## 四、对接成员 C（后端接口）

### 4.1 检索函数签名

```python
from search_engine import SearchEngine, SearchResult

# 初始化
engine = SearchEngine(
    vectors: np.ndarray,      # (n_cells, n_dims) float32
    metadata: pd.DataFrame,   # (n_cells, *) 细胞元数据
    method: str = "faiss"     # "brute" / "faiss" / "hnsw"
)

# 构建索引
engine.build_index(nlist=100, nprobe=10)

# 检索接口
result: SearchResult = engine.search(
    query: np.ndarray,        # (n_queries, n_dims) 或 (n_dims,)
    k: int = 10,              # 返回 top-k
    filters: dict = None,     # 条件过滤 {"cell_type": "hepatocyte"}
    metric: str = "cosine"    # "cosine" / "euclidean"
)

# 按细胞索引查询
result = engine.search_by_cell_index(
    cell_index: int,          # 细胞在数组中的位置 (0-based)
    k: int = 10,
    filters: dict = None,
    metric: str = "cosine"
)

# 按自定义向量查询
result = engine.search_by_vector(
    vector: np.ndarray,       # (n_dims,) 自定义向量
    k: int = 10,
    filters: dict = None,
    metric: str = "cosine"
)
```

### 4.2 SearchResult 数据结构

```python
@dataclass
class SearchResult:
    indices: np.ndarray          # (n_queries, k) 匹配细胞索引
    distances: np.ndarray        # (n_queries, k) 余弦相似度 (1=完全匹配)
    metadata: List[List[Dict]]   # 每个查询结果对应的细胞元数据
    query_time_ms: float         # 查询耗时（毫秒）
    method: str                  # 检索方法名称
```

### 4.5 后端集成示例

```python
def handle_search_request(query_vector, k, cell_type=None, age_group=None):
    """供 Flask/FastAPI 调用的检索函数"""
    import numpy as np
    import pandas as pd
    from search_engine import SearchEngine

    # 全局初始化（启动时执行一次）
    if not hasattr(handle_search_request, "engine"):
        vectors = np.load("cell_vectors.npy")
        metadata = pd.read_csv("cell_metadata.csv", index_col=0)
        handle_search_request.engine = SearchEngine(vectors, metadata, method="faiss")
        handle_search_request.engine.build_index(nlist=100)

    engine = handle_search_request.engine

    # 构建过滤条件
    filters = {}
    if cell_type:
        filters["cell_type"] = cell_type
    if age_group:
        filters["AgeGroup"] = age_group
    if not filters:
        filters = None

    result = engine.search(query_vector, k=k, filters=filters)

    # 返回 JSON 可序列化的结果
    return {
        "results": result.metadata[0],
        "query_time_ms": result.query_time_ms,
        "method": result.method
    }
```

## 八、预处理流水线详情

```
原始数据 (69,032 cells × 32,397 genes)
    │
    ▼ QC: 过滤低质量细胞
    │   - min_genes=200, max_pct_mt=20%
    │   - 保留 61,245 细胞, 25,403 基因
    │
    ▼ 标准化: normalize_total(target_sum=10,000)
    │
    ▼ 对数变换: log1p
    │
    ▼ 高变基因筛选: top 2,000 HVGs (seurat)
    │
    ▼ 缩放: Z-score, clip max=10
    │
    ▼ PCA: 64 主成分 (解释方差 36.25%)
    │
    输出: (61245, 64) PCA 向量
```

> **注意**: 检索模块默认使用数据自带的 30 维 PCA（速度快，覆盖全量 69K 细胞）。如需使用新算的 64 维向量，将 `main.py` 中 `use_vectors` 改为 `pca_vectors_new` 即可。

## 九、注意事项

1. **向量格式约定**: 所有向量均为 float32，已 L2 归一化，可直接用于 FAISS 内积搜索
2. **条件过滤策略**: 当过滤后候选集 <30% 总量时，自动回退暴力搜索以保证准确率
3. **数据只读**: 除预处理模块外，所有操作不修改原始 `liver.h5ad` 文件
4. **依赖版本**: 建议 Python ≥3.10, scanpy ≥1.9, faiss-cpu ≥1.7
